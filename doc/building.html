Building Cirsim and Kmap

To build Cirsim and Kmap you will need to have Node.js installed. To add the dependencies, use this command:

npm install

Then to build:

npm run build:all

This will build both the development and distribution versions. If you want to build only the development version (build.js):

npm run build:dev

For only the production version:

npm run build:prod

Support is included for running the development server and hot-swapping. You will need to edit webpack.dev.js and change the host name you are using. I used cirsim.localhost.

THere is a file cirsim.inc.php that is included in <head> of all of the demonstrations. It uses dist/build.js if it is present and the CDN if it is not.

<?php
if(file_exists(__DIR__ . '/../dist/build.js')) {
    echo <<<HTML
<script src="dist/build.js"></script>
HTML;
} else {
    echo <<<HTML
<script src="https://unpkg.com/cirsim/dist/cirsim.min.js"></script>
<script src="https://unpkg.com/kmap/dist/kmap.min.js"></script>
HTML;
}
